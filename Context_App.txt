equipment exchange, cost estimation) with dual-bot architecture (TG + MAX),
React frontend, FastAPI backend, SQLite database.

Stack: FastAPI + SQLite (WAL) | React + Vite + Tailwind + framer-motion | aiogram (TG) + maxapi (MAX)
Graph: 797 nodes, 820 edges, 128 communities (graphify)
